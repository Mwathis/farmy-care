LocalCast

LocalCast is a simple web application that will provide you with real time current weather information of any any area you would like in the world.

You will be able to check for the following weather conditions:

  1. Humidity Level
  2. Temperature
  3. Wind Direction
  4. Atmospheric Pressure
  5. Wind speed
