Farmy Care

Farmy care is a web application that will provide farmers with real time weather information from there local areas.This will help them effectively plan their farm activities.

The farmers will be able to check for the following weather conditions:

  1. Humidity Level
  2. Temperature
  3. Liquid precipitation rate
  4. Atmospheric Pressure
  5. Wind speed
